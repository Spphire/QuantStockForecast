# 实战优化 TODO 看板

更新时间：2026-03-23（Asia/Shanghai）

## 使用方式

每天开工时可直接问：

- `我们 TODO 看板现在有哪些任务？`
- `按优先级给我挑 2-3 个今天做`
- `把 <ID> 状态改成 in_progress / done`

状态约定：

- `backlog`：待排期
- `in_progress`：进行中
- `blocked`：有阻塞
- `done`：已完成（待观察）
