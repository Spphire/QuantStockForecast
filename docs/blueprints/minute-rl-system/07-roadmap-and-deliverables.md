# 07 里程碑与交付清单

## 1. 里程碑

1. M0（1-2天）：契约冻结、配置模板、审计字段框架。
2. M1（3-5天）：分钟数据抓取与特征落盘稳定运行。
3. M2（4-7天）：分钟 expert 与 ensemble 全链路可用。
4. M3（3-5天）：分钟 `white-box -> execution` 基线可运行（无 RL 在线更新）。
5. M4（4-7天）：`IQL -> SAC` challenger shadow 闭环。
6. M5（1-2周）：双 paper 小资金 A/B。
7. M6（2-4周）：等权 A/B，产出晋级决策报告。

## 2. 分工建议

1. SA-Data：分钟数据与特征。
2. SA-Experts：分钟模型训练/推理。
3. SA-RL：状态、奖励、replay、在线训练。
4. SA-RiskExec：风控桥接、执行编排、watchdog。
5. SA-Ops：SLO、告警、简报、runbook。

## 3. 每阶段退出条件（Exit Criteria）

1. M1 退出：连续 5 个交易日无断档，数据新鲜度达标，schema 无漂移。
2. M2 退出：预测文件契约全通过，信号链路可回放。
3. M3 退出：基线拒单率/超时挂单率不超阈值，账本闭环完整。
4. M4 退出：shadow 指标稳定，reward attribution 与梯度健康全通过。
5. M5/M6 退出：满足晋级统计门槛或明确拒绝晋级并产出复盘。

## 4. 首版暂缓项

1. 全深度订单簿模拟。
2. 亚秒级微结构建模。
3. RL 直接控制逐股逐单。
4. Decision Transformer 线上主力化。
5. 多市场联合在线训练。
