# Crawlers

`data_module/crawlers` 是预留的网页爬虫模块，用来补充结构化接口拿不到的信息。

## 计划职责

- 新闻抓取
- 公告抓取
- 研报抓取
- 舆情与论坛文本抓取
- 财经网站页面解析

## 为什么它不是当前第一优先级

当前项目的主线还是：

- 先把行情数据链路跑通
- 先验证横截面收益预测和风控组合逻辑

因为对于日线股票研究来说，**先有稳定的价格和成交数据**，往往比一开始就上复杂爬虫更重要。

## 更适合它的使用场景

- 做新闻情绪增强
- 做公告事件驱动
- 构建公告发布时间和事件标签
- 做多模态因子

## 推荐未来结构

- `news/`
- `announcements/`
- `research_reports/`
- `forums/`
- `parsers/`

## 当前状态

- 规划中
- 尚未接入正式抓取脚本

## 建议

如果后面开始做爬虫，优先让它输出“结构化中间表”，而不是直接耦合模型输入。这样可以让 `cleaning` 和 `features` 层继续保持清晰边界。
